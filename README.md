## 🌟 Proyectos Principales

### 🎓 [SIPAD](https://github.com/enmanuelgs) — Sistema Predictivo de Abandono Escolar *(Tesis)*
Sistema de analítica avanzada para predecir la deserción escolar a escala nacional en la República Dominicana. Integra datos de múltiples instituciones como **MINERD, ONE, CONANI y MISPAS**.
- 🏗️ **Arquitectura**: Clean Architecture & Repository Pattern.
- ⚙️ **Stack**: C# ASP.NET Core · PostgreSQL · Entity Framework Core 8.0 · CsvHelper.
- 📦 **Detalles**: Implementación de 12 servicios ETL, orquestador de pipeline y 8 controladores REST.

### 🏛️ [SEGMUN](https://github.com/enmanuelgs/SEGMUN) — Sistema de Gestión para Modelos UN
Solución tecnológica integral para la administración y evaluación de Modelos de Naciones Unidas. Automatiza la gestión de delegados, comités y garantiza precisión en las métricas de desempeño mediante una interfaz moderna.
- ⚙️ **Stack**: .NET 9 · C# · Entity Framework Core · PostgreSQL · React.
- 📋 **Impacto**: Optimización de la lógica de evaluación y seguimiento en tiempo real de pases de lista y participaciones.
